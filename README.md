# Entera KP — статика для Vercel

Здесь лежит **готовый к деплою** набор: `index.html` (дека 9 слайдов) + `assets/logo-entera-from-site.svg`.

## Онлайн-ссылка

После деплоя корень проекта (`/`) отдаёт презентацию. Прямая ссылка вида `https://<project>.vercel.app/`.

## PDF

1. Кнопка **«Скачать PDF (9 слайдов)»** открывает вкладку с **всеми слайдами** в режиме печати (`?print=1`).
2. В новой вкладке нажмите **«Открыть печать / PDF»** или **Ctrl+P** (Mac: **⌘+P**).
3. Выберите принтер **«Сохранить как PDF»** / **Save as PDF**.

**Рекомендуемое имя файла:** `Entera-KP-9slides.pdf` (совпадает с `<title>` страницы — многие браузеры подставляют его в диалоге сохранения).

Серверный «скачать .pdf одной кнопкой без диалога» здесь **не** подключён: для статики на Vercel надёжнее печать в PDF из браузера (шрифты Gilroy с CDN уже подтянуты).

## Деплой на Vercel

```bash
cd /home/dev/POS/projects/pdf-recognition-construction/docs/commercial-pdf/vercel-site
npx vercel login   # один раз
npx vercel --prod  # выдать публичный URL
```

Каталог можно переименовать в проекте Vercel или привязать к своему домену в настройках Vercel.

Папка `.vercel` создаётся CLI и указана в `.gitignore` — её можно не коммитить или коммитить для фиксации project id (на усмотрение).
